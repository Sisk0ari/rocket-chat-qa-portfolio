# QA Portfolio — Rocket.Chat Testing

## О проекте

Выпускное тестовое задание в рамках обучения 
в АНО «Школа 21» (Сбер).

**Объект тестирования:** Rocket.Chat — 
open-source платформа для корпоративных коммуникаций 
(веб-интерфейс + API).

## Что было сделано

- Анализ требований и составление тест-плана
- Функциональное тестирование основных сценариев: 
  авторизация, чаты, каналы, права доступа, настройки
- Написание тест-кейсов и чек-листов
- Оформление баг-репортов
- API-тестирование через Postman
- Работа с тестовыми данными

## Структура репозитория

| Папка | Содержание |
|-------|-----------|
| `code-samples` | Примеры запросов / скриптов |
| `data-samples` | Тестовые данные |
| `datasets` | Наборы данных для тестирования |
| `materials` | Документация, требования |
| `src` | Исходные материалы |

## Инструменты

Postman · Chrome DevTools · GitHub · 
REST API · HTTP/HTTPS · JSON

## Навыки, продемонстрированные в проекте

- Составление тест-кейсов и чек-листов
- Оформление баг-репортов
- API-тестирование (Postman)
- Работа с документацией продукта
- Тестирование прав доступа и ролей пользователей
